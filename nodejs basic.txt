Node.js
Node.js is an open-source, cross-platform JavaScript runtime environment that can run on Windows, Linux, Unix, macOS, and more. Node.js runs on the V8 JavaScript engine, and executes JavaScript code outside a web browser.

npm package 
NPM, or Node Package Manager, is the default package manager for the Node.js JavaScript runtime environment. It's used to install, develop, discover, and publish Node programs.

npm init => you can install automatically package.json =>  (it used to version and dependencies)
you can install some programs like express js, mongoDb and more, 
npm install <package name>

Express 
Fast, unopinionated, minimalist web framework for Node.js

Install Express as a dependency for your project using the npm install express command. This command installs the Express module and adds it to your project's node_modules folder.
=> npm install express

Nodemon 
nodemon is a used to run a program automatically run server or program 

=> nodemon install -g nodemon (this command is used to install nodemon globla version)

After that you can install express and mongodb => npm install express && mongodb

Dotenv 
dotenv is a popular Node.js module that helps manage environment variables in your Node.js applications. Environment variables are key-value pairs that can be used to configure your application, and they are often used for sensitive information like API keys, database credentials, or other configuration settings.

=> npm install dotenv
